┌─ FEATURES ─┐

  • Desktop as stage: wallpapers glitch, UI "breaks" a little.
  • Inbox storytelling: friend, warnings, and spam threads push the plot.
  • Visual escalation (4 phases): cute comic pups → photo-real → warped → dread tableaux.
  • Timer pressure (Stage 3): fail = in-story "fail mail".
  • Finale switch (Stage 4): scene-hunt to make you stare.
  • Optional "JUMPSCARE" toggle in Settings.
  • Stage-based audio: cheerful → hushed → breaths/static → near-silence.
